## SYNTAX
- **COMMANDS** | _All_ **COMMANDS** **inside** with **()**
  - **(** _function_ _args_ **)**
```clojure
 (+ 1 2 )
```
- **MAIN PROJECT***
```clojure
(defn -main
  "Comment"
  [& args]
  (commands)
  )
  ```
- **CREATE VARIABLE**
  - (**def** _varName_ **value**)
```clojure
(def age 23)
```
**Create FUNCTIONS**
```clojure
(defn name [param] (Math/pow x 2))
```
- **CREATE NAMESPACE**
```clojure
(ns packageName)
```
- **Call N functions** | Use "**do**"
```clojure
(do
  (fn)
  (fn)
  )
```


### CONDITIONALS
